# LN Visual

## 项目名称
《李宁品牌 AI 视觉规范系统》

本项目用于沉淀李宁品牌在 AI 视觉生成中的基础规范、品类方向、活动场景、节日场景和 Codex 工作流。它不是单张图片的提示词合集，而是一套可长期维护、可反复调用的品牌视觉规则库。

## 项目目标
建立一套适合设计师、运营、电商和内容团队使用的 AI 视觉规范系统，让后续生成海报、社群图、电商图、素材图、3D 模型、视频分镜和动态视觉时，有统一的品牌判断依据。

这套系统重点服务以下目标：

- 保持李宁品牌识别统一
- 强化专业运动、中国运动品牌、年轻化表达
- 提升画面的速度感、能量感、功能科技感
- 建立东方运动美学的视觉表达方式
- 适配电商营销、社群传播和内容种草场景
- 帮助 AI 生成结果更稳定、更像李宁、更适合实际使用

## 适用场景
本项目适合用于以下 AI 视觉任务：

- 品牌海报方向整理
- 电商主图和活动 KV 视觉规划
- 社群海报、朋友圈图、小红书封面方向整理
- 产品素材图、氛围图、背景图生成前的风格定义
- 3D 模型、材质、场景风格的前期视觉设定
- 视频分镜、动态视觉、开屏动画的视觉方向规划
- 大促、节日、新品首发、奥莱专场等营销场景的视觉统一

## 文件夹结构说明
当前项目的核心内容放在：

```text
Li-Ning-Brand-AI-Visual-Guidelines
```

### 01_Brand-Foundation
品牌基础规范。

这里定义整套系统的底层方向，包括品牌 DNA、品牌关键词、品牌禁区、品牌语气与文案风格。后续所有视觉内容都应该先对齐这一部分。

### 02_Category-Visual-Guidelines
品类与系列视觉规范。

这里定义不同运动品类和产品系列的视觉方向，例如跑步、篮球、儿童、训练、户外，以及反伍、韦德、飞电、中国李宁、COUNTERFLOW、金标等系列。做具体产品视觉时，优先查这里。

### 03_Campaign-Visual-Guidelines
大促视觉规范。

这里定义电商活动和营销节点的视觉方向，例如 618、双11、年货节、会员日、直播大促、新品首发、奥莱专场等。做电商活动图时，优先查这里。

### 04_Festival-Visual-Guidelines
节日视觉规范。

这里定义节日和节气营销的视觉方向，例如春节、端午、中秋、国庆、七夕、圣诞、元旦、母亲节、儿童节等。做节日海报和社群图时，优先查这里。

### 05_Codex-Workflow
Codex 工作流。

这里记录如何让 Codex 使用这套视觉规范库，包括常用指令、使用说明和“李宁视觉总监”角色设定。它是日常调用这套系统的操作入口。

## 视觉规范调用优先级

本项目支持根据需求类型自动调用视觉规范：

- 品类任务优先调用品类规范。
- 活动任务优先调用活动规范。
- 节日任务优先调用节日规范。
- 品牌基础规范作为底层兜底。

处理视觉任务时，应先识别用户需求中明确出现的品类、活动、节日；未提供的信息标记为“需求未提供”，不要自动补全。

完整规则见：

```text
05_Codex-Workflow/Visual-Guideline-Routing-Rules.md
```

## 后续维护方式
这套规范库建议按“小步更新”的方式维护。

每次只完善一个文件或一个场景，不要一次性大改全部内容。比如今天只完善“跑步品类”，明天只完善“618 大促”，这样更容易检查和回退。

推荐维护节奏：

1. 先完善品牌基础规范
2. 再完善最常用的品类规范
3. 再完善电商活动和节日场景
4. 最后补充 3D、视频、动态视觉的专用表达方式

每次改完一批内容后，建议用 Git 保存一个版本，方便以后回到之前的状态。

## Codex 使用方法
使用 Codex 时，不要只说“帮我生成一张图”。更好的方式是让 Codex 先读取对应规范，再整理视觉方向。

推荐说法：

```text
请读取 LN Visual 项目中的品牌基础规范和篮球品类规范，帮我整理一张李宁篮球新品海报的视觉方向。
```

也可以这样说：

```text
请参考 618 大促视觉规范和跑步品类规范，帮我规划一张电商首页 KV 的画面结构。
```

如果是社群图，可以这样说：

```text
请参考春节视觉规范和运动生活品类规范，帮我整理一张适合社群发布的李宁节日海报方向。
```

## AI 出图使用流程
后续使用这套系统进行 AI 跑图时，建议按下面流程进行：

1. 明确要做什么
   例如海报、社群图、电商图、素材图、3D 模型、视频分镜或动态视觉。

2. 明确使用场景
   例如电商首页、商品主图、社群传播、小红书封面、直播间封面、节日活动页。

3. 选择对应规范文件
   先根据需求类型判断品类、活动或节日；品类任务优先看品类规范，活动任务优先看活动规范，节日任务优先看节日规范，品牌基础规范作为底层调性和禁区兜底。

4. 让 Codex 整理视觉方向
   先输出画面气质、构图方向、色彩方向、产品呈现方式、文案语气和禁区。

5. 再进入提示词阶段
   提示词应基于规范整理，不要脱离品牌直接堆关键词。

6. 生成后进行判断
   检查画面是否像李宁、是否专业运动、是否年轻有能量、是否适合电商或社群使用。
