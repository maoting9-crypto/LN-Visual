# LN Visual｜李宁品牌 AI 视觉规范系统

## 项目名称

《李宁品牌 AI 视觉规范系统》

本项目用于沉淀李宁品牌在 AI 视觉生成中的基础规范、品类方向、活动场景、节日场景和 Codex 工作流。它不是单张图片的提示词合集，而是一套可长期维护、可反复调用的品牌视觉规则库。

## 项目目标

建立一套适合设计师、运营、电商和内容团队使用的 AI 视觉规范系统，让后续生成海报、社群图、电商图、素材图、3D 模型、视频分镜和动态视觉时，有统一的品牌判断依据。

重点目标：

- 保持李宁品牌识别统一。
- 强化专业运动、中国运动品牌、年轻化表达。
- 提升画面的速度感、能量感、功能科技感。
- 建立东方运动美学的视觉表达方式。
- 适配电商营销、社群传播和内容种草场景。
- 帮助 AI 生成结果更稳定、更像李宁、更适合实际使用。

## 文件夹结构说明

### 01_Brand-Foundation

品牌基础规范。

这里定义整套系统的共同基础层，包括品牌 DNA、品牌关键词、品牌禁区、品牌语气与文案风格。后续所有视觉任务都必须参考这一部分，但它会根据任务类型与品类、活动、节日规范进行加权融合。

### 02_Category-Visual-Guidelines

品类与系列视觉规范。

这里定义不同运动品类和产品系列的视觉方向，例如跑步、篮球、儿童、训练、户外、运动生活，以及反伍、韦德、飞电、中国李宁、COUNTERFLOW、金标等系列。做具体产品视觉时，需要根据用户明确提供的品类调用这里。

### 03_Campaign-Visual-Guidelines

活动与大促视觉规范。

这里定义电商活动和营销节点的视觉方向，例如 618、双11、年货节、会员日、直播大促、新品首发、奥莱专场等。做电商活动图时，需要根据用户明确提供的活动调用这里。

### 04_Festival-Visual-Guidelines

节日视觉规范。

这里定义节日和节气营销的视觉方向，例如春节、端午、中秋、国庆、七夕、圣诞、元旦、母亲节、儿童节等。做节日海报和社群图时，需要根据用户明确提供的节日调用这里。

### 05_Codex-Workflow

Codex 工作流。

这里记录如何让 Codex 使用这套视觉规范库，包括常用指令、使用说明、视觉规范调用规则和“李宁视觉总监”Skill 说明。

## 视觉规范加权融合

本项目支持“视觉规范加权融合”：

- 品类任务 = 品类规范 + 品牌基础规范。
- 活动任务 = 活动规范 + 品牌基础规范。
- 节日任务 = 节日规范 + 品牌基础规范。
- 多维度任务 = 主主题规范 + 次主题规范 + 品类规范 + 品牌基础规范。

品牌基础规范始终参与视觉判断，不只是兜底。

基础权重：

- 单一品类任务：品类规范 60% + 品牌基础规范 40%。
- 单一活动任务：活动规范 60% + 品牌基础规范 40%。
- 单一节日任务：节日规范 60% + 品牌基础规范 40%。
- 活动 + 品类任务：活动规范 50% + 品类规范 30% + 品牌基础规范 20%。
- 节日 + 品类任务：节日规范 50% + 品类规范 30% + 品牌基础规范 20%。
- 节日 + 活动任务：主主题规范 50% + 次主题规范 30% + 品牌基础规范 20%；主主题不明确时，节日规范 50% + 活动规范 30% + 品牌基础规范 20%。
- 节日 + 活动 + 品类任务：主主题规范 45% + 次主题规范 25% + 品类规范 15% + 品牌基础规范 15%；主主题不明确时，节日规范 45% + 活动规范 25% + 品类规范 15% + 品牌基础规范 15%。
- 仅品牌任务：品牌基础规范 100%。

完整规则见：


## 后续维护方式

这套规范库建议按“小步更新”的方式维护。

每次只完善一个文件或一个场景，不要一次性大改全部内容。比如今天只完善“跑步品类”，明天只完善“618 大促”，这样更容易检查和回退。

每次改完一批内容后，建议用 Git 保存一个版本，方便以后回到之前的状态。

## Codex 使用方法

使用 Codex 时，不要只说“帮我生成一张图”。更好的方式是让 Codex 先识别需求，再按加权融合规则调用对应规范。

推荐说法：

```text
请调用 li-ning-visual-director Skill，先识别本次需求中的品类、活动、节日和主主题，再按视觉规范加权融合规则读取对应规范，输出本次调用权重和融合后的视觉方向。
```

## AI 出图使用流程

1. 明确要做什么：例如海报、社群图、电商图、素材图、3D 模型、视频分镜或动态视觉。
2. 明确使用场景：例如电商首页、商品主图、社群传播、小红书封面、直播间封面、节日活动页。
3. 识别需求维度：判断用户是否明确提供品类、活动、节日和主主题。
4. 按加权融合规则调用规范：专项规范决定视觉方向，品牌基础规范始终参与品牌一致性控制。
5. 让 Codex 整理视觉方向：先输出画面气质、构图方向、色彩方向、产品呈现方式、文案语气和禁区。
6. 再进入提示词阶段：提示词应基于规范整理，不要脱离品牌直接堆关键词。
7. 生成后进行判断：检查画面是否像李宁、是否专业运动、是否年轻有能量、是否适合电商或社群使用。
