# Common Commands

## V1.0 常用指令

下面这些句子可以直接复制给 Codex 使用。

## 按加权融合规则调用视觉规范

```text
请调用 li-ning-visual-director Skill，先读取用户需求中明确出现的品类、活动、节日，再按视觉规范加权融合规则调用对应规范。品牌基础规范不是兜底，而是每次都参与视觉判断。请输出本次调用的规范文件、调用权重和最终融合后的视觉方向。不要自动补全需求中未提供的信息。
```

## 读取品牌基础规范

```text
请读取 LN Visual 项目中的品牌基础规范，并结合官方 VI 校准规则，帮我总结这次出图应该遵守的李宁品牌视觉原则。
```

## 生成海报提示词

```text
请根据李宁品牌基础规范、用户明确指定的品类 / 活动 / 节日规范，以及官方 VI 校准规则，生成一条中文 AI 出图提示词。要求包含画面主体、动作、场景、光影、构图、色彩、Logo 留白、文字后期排版建议和 negative prompt。
```

## 检查海报

```text
请根据李宁品牌 AI 视觉规范检查这张海报：是否符合本次规范调用权重、是否符合李宁专业运动品牌调性、Logo 是否安全、文字是否清晰、产品是否突出、是否存在廉价促销或品牌禁区问题。
```

## 生成系列海报

```text
请生成一组李宁系列海报提示词，要求版式一致、比例一致、Logo 区域一致、标题区域一致、画面风格统一，但每张图的人物动作、产品表达和场景根据对应品类或活动变化。
```

## 电商 KV

```text
请参考李宁线上店铺形象规范、对应活动规范和品牌基础规范，帮我规划一张电商 KV：画面干净、产品主体清晰、活动主题和利益点分层、重要元素避开裁切和遮挡区域。
```

## 视频分镜

```text
请根据李宁品牌视觉规范，帮我写一个短视频分镜。要求前三秒出现产品或强动作，画面角落预留品牌角标，结尾可参考黑底白标的品牌收束方式。
```

## 3D 视觉

```text
请根据李宁品牌视觉规范，帮我生成一个 3D 产品展示方向。要求材质真实、产品比例准确、科技结构清晰、背景不抢主体，并预留 Logo 和文字区域。
```
