# Codex User Guide

## V1.0 官方 VI 校准版目标
本文件说明如何让 Codex 使用这套《李宁品牌 AI 视觉规范系统》。

现在项目已经结合两份官方 VI 资料：

- 《李宁品牌视觉规范.pdf》
- 《2022-03-24 LN线上店铺形象识别与应用规范.pdf》

## 基本使用步骤
1. 先说明你要做什么：海报、社群图、电商图、素材图、3D 模型、视频分镜或动态视觉。
2. 再说明使用场景：社群传播、电商首页、直播封面、活动 KV、产品素材、视频角标等。
3. 指定参考模块：品牌基础、品类规范、大促规范、节日规范。
4. 让 Codex 先整理视觉方向，再生成提示词。
5. 正式投放前，再检查 Logo、颜色、字体、口号和平台尺寸是否符合官方规范。

## 官方 VI 使用原则
- Logo 不建议由 AI 直接生成，应该后期使用官方资产置入。
- 李宁红、黑、白、灰等色彩可以作为 AI 风格参考，最终输出需以官方资产为准。
- 官方品牌主张“一切皆有可能”可作为品牌语境，但正式使用需遵循官方口号规范。
- 线上店铺画面要保持干净、清晰、阅读方便。
- 大促画面可以有冲击力，但应避免高饱和撞色和廉价模板感。

## 推荐提问方式
```text
请根据 LN Visual 项目中的品牌基础规范、跑步品类规范和官方 VI 校准规则，帮我生成一张李宁社群海报的中文 AI 出图提示词。
```

```text
请参考李宁线上店铺规范，帮我规划一张 3:4 电商 KV，要求产品主体清楚、Logo 区域留白、黑白灰主视觉、李宁红局部点缀。
```

```text
请根据篮球品类和李宁官方 VI 规则，帮我检查这张海报是否存在 Logo、色彩、文字或运动调性问题。
```

## AI 出图前检查
- 是否明确画面比例？
- 是否明确品类？
- 是否明确使用场景？
- 是否预留 Logo 区域？
- 是否说明文字后期排版？
- 是否避免让 AI 直接生成中文大段文字？
- 是否避免编造官方科技名和官方口号？
