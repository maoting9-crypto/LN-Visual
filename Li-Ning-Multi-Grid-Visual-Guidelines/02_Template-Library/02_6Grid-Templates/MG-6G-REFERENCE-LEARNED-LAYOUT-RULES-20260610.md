# MG-6G-REFERENCE-LEARNED-LAYOUT-RULES-20260610

## 模板定位

这是从 2026-06-10 六宫格参考素材中学习出的布局规则集合。

它不是风格模板，不记录、不复用、不要求照搬原图品牌、配色、字体、背景、人物、文案、光影、装饰元素或具体视觉风格。

## 适用宫格类型

- 六宫格
- 3x2 横向布局
- 2x3 竖向布局
- 默认输出一张完整六宫格海报，不拆成单格图片

## 可复用布局规则

### 1. 左侧主题 + 右侧产品矩阵型

- 左侧 1-2 格作为主题区或活动入口。
- 右侧 3-4 格作为产品卡或人物 / 产品组合。
- 适合活动推广、产品推荐、朋友圈横向展示。

### 2. 主题 / 二维码 + 多产品卡型

- 主题区、二维码区和产品区分开。
- 二维码可放左下、右下、下中或侧边完整格。
- 产品卡围绕主题和二维码展开。

### 3. 节日主题区 + 产品 / 人物组合型

- 一侧作为节日或活动情绪入口。
- 另一侧以产品卡、人物或场景进行承接。
- 节日元素只作为模块关系参考，不复用原装饰。

## 产品摆放规则

- 产品卡应保持统一尺寸、统一边距、统一信息层级。
- 横向六宫格中，产品卡需要保持左右视觉重量平衡。
- 如果出现权益 / 价格区，只记录其位置和层级，不复用原文字。

## 模特素材运用规则

- 模特可占一格或跨两格。
- 模特方向应引导视线进入产品区或二维码区。
- 不复用参考素材原人物。

## 场景与背景延展规则

- 六宫格可采用分区背景，不强制整图连续。
- 如果使用连续背景，必须保证主题区、产品区和二维码区仍然清晰。
- 不复用参考图原背景、光影和氛围。

## 二维码区域规则

- 二维码必须独占完整格。
- 不贴边角，不跨格，不压住产品、人物或标题。
- 正式发布二维码建议后期替换。

## 裁切规则

- 横向六宫格需要保证左右视觉平衡。
- 主题区不可被产品卡挤压。
- 产品、人物、二维码和标题不能被格线切坏。

## 禁止事项

- 不跨宫格类型套用本规则。
- 不照搬原图品牌、配色、字体、背景、人物、文案、光影、装饰元素。
- 不在最终提示词中写本地路径。
