# MG-6G-QR-PROMO-MATRIX-001

## 模板编号

MG-6G-QR-PROMO-MATRIX-001

## 宫格形式

六宫格，通常为 3x2 横向结构，也可转译为 2x3 竖向结构。

## 模板定位

这是六宫格二维码引流促销矩阵布局模板，只记录信息布局和模块关系，不记录、不复用、不要求照搬参考图的原图风格、配色、背景、人物、光影、品牌气质或装饰元素。

## 整体布局结构

- 整体由 6 个等分格组成。
- 主标题 / 活动主题通常位于上中、左上或跨上方信息区。
- 产品卡片围绕主题区排列，形成 3-5 个商品推荐位。
- 二维码作为独立模块出现，通常位于右上、右下、下中或侧边稳定转化区。
- 价格 / 权益信息跟随各产品卡片底部或主题格中部，不跨越主要产品主体。

## 每格功能

3x2 默认编号：

- 第 1 格：主推产品、人物 / 场景区或活动标题延展。
- 第 2 格：主标题 / 活动主题 / 核心权益。
- 第 3 格：二维码独占预留格或重点产品格。
- 第 4 格：产品推荐卡。
- 第 5 格：产品推荐卡或权益补充格。
- 第 6 格：产品推荐卡、二维码独占预留格或收口信息格。

2x3 默认编号：

- 第 1 格：主标题 / 活动主题。
- 第 2 格：二维码独占预留格或重点产品格。
- 第 3 格：产品推荐卡。
- 第 4 格：产品推荐卡。
- 第 5 格：产品推荐卡或权益补充格。
- 第 6 格：收口信息、二维码独占预留格或产品推荐卡。

## 主标题区位置

优先位于上方信息区或第 2 格，需保持手机端快速识别。

## 产品区位置

产品区通常分布在下排和侧边格位，产品主体居中，商品名、货号、价格 / 权益区按卡片层级固定。

## 二维码区位置

二维码必须作为独立模块预留，不跨格，不压住产品、人物、主标题、价格或权益。正式发布时后期替换真实二维码。

## 价格 / 权益区位置

价格 / 权益区跟随产品卡片底部或主题信息区，优先保持同一系列的高度和对齐关系。

## 人物 / 场景区位置

人物 / 场景区如存在，通常作为第 1 格或上方主视觉辅助区，用于建立品类场景；模板只记录该区位置关系，不记录原图人物形象。

## 信息层级关系

主标题 / 活动主题 > 核心权益 / 二维码入口 > 产品主体 > 商品名 / 货号 > 价格 / 权益补充。

## 适合的运营需求类型

- 朋友圈促销。
- 奥莱专场。
- 618 / 双11 / 大促节点。
- 多产品推荐。
- 扫码引流。
- 门店导购转发。

## 禁止项

- 不照搬参考图原配色。
- 不照搬参考图原背景。
- 不照搬参考图原人物。
- 不照搬参考图原风格。
- 不照搬参考图原光影。
- 不照搬参考图原装饰元素。
