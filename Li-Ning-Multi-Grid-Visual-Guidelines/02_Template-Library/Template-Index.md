# Template Index

本文件记录当前多宫格模板库中可复用的布局规则。

## 模板库定位

`02_Template-Library` 是“多宫格布局规则库”，不是“参考图风格复刻库”。

模板库只记录从参考素材中学习出的：

- 宫格类型
- 布局规则
- 模块关系
- 产品摆放规则
- 元素摆放规则
- 模特与场景运用规则
- 背景延展规则
- 二维码区域规则
- 裁切规则
- 发布规则

模板库不记录、不复用：

- 原图品牌
- 原图配色
- 原图字体
- 原图背景
- 原图人物
- 原图文案
- 原图光影
- 原图装饰元素
- 原图具体视觉风格

## 宫格类型硬规则

宫格类型是第一硬条件，不允许跨宫格类型直接套用结构。

- 九宫格任务只能使用九宫格模板。
- 六宫格任务只能使用六宫格模板。
- 四宫格任务只能使用四宫格模板。
- 三宫格任务只能使用三宫格模板。
- 双图组合任务只能使用双图组合模板。

## 输出表达要求

最终提示词不得写：

- 参考 G:
- 按本地路径图片生成
- 照搬参考图
- 复刻参考图
- 完全复刻参考图视觉风格

应改写为：

```text
从多宫格参考素材中学习布局原理和素材运用规则，并转写为外部 AI 平台可理解的完整描述。
```
