# Template Index

本文件记录当前多宫格模板库中可复用的布局模板。

## 模板库定位

`02_Template-Library` 是“参考图信息布局与模块关系模板库”，不是“照搬参考图风格的模板库”。

模板库只记录从 `G:\Reference-Library\01_Multi-grid-reference` 中提取的宫格结构、信息布局、模块位置、每格功能、主标题区、产品区、二维码区、价格 / 权益区、人物 / 场景区和信息层级关系。

模板不记录、不复用、不要求照搬原图配色、原图背景、原图人物、原图风格、原图装饰元素、原图光影或原图品牌气质。

## 强锁定模式说明

当用户要求“严格按照参考图来做”时，模板进入强锁定模式。

- 固定结构层：布局结构、模块关系、各区块位置、产品摆放逻辑、二维码位置、信息层级、拼接关系。
- 可变视觉层：配色、字体设计、图片内元素内容、整体风格调性、人物素材、背景。

强锁定模式下，风格不继承，结构必须继承。模板输出必须明确区分固定结构层和可变视觉层。
