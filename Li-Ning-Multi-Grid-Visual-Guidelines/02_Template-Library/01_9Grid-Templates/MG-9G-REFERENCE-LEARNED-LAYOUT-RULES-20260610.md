# MG-9G-REFERENCE-LEARNED-LAYOUT-RULES-20260610

## 模板定位

这是从 2026-06-10 九宫格参考素材中学习出的布局规则集合。

它不是风格模板，不记录、不复用、不要求照搬原图品牌、配色、字体、背景、人物、文案、光影、装饰元素或具体视觉风格。

## 适用宫格类型

- 九宫格
- 3x3
- 完整 1:1 方形多宫格海报
- 默认输出一张完整九宫格海报，不拆成 9 张单独图片

## 可复用布局规则

### 1. 标题 + 权益 + 二维码 + 产品矩阵型

- 主标题位于左上、上中或中心格。
- 权益 / 价格 / 活动信息位于右上、上中或主题格旁边。
- 二维码独占中心、中右、下中或中左完整格。
- 产品卡围绕二维码或主题格形成矩阵。
- 适合奥莱、618、双11、会员日、多产品推荐。

### 2. 人物 / 场景跨格 + 产品卡型

- 人物或场景跨 2-4 格形成主视觉。
- 主标题独立占 1 格或半格区域。
- 产品卡放在右侧或下排。
- 二维码独占一格，不压人物和产品。
- 适合品宣、活动转化、运动场景表达。

### 3. 中心二维码 / 中心主题 + 周围产品型

- 中心格承担二维码或主题入口。
- 周围 8 格承载产品、人物、场景或权益信息。
- 适合社群转化、朋友圈推广和产品矩阵推荐。

### 4. 全产品矩阵型

- 每格一个产品或产品组合。
- 所有产品卡统一图像比例、信息层级和边距。
- 可设置一格作为主题，其他格展示产品。
- 适合多 SKU 推荐、货盘展示、奥莱专场。

### 5. 主题区 + 模特区 + 产品区混合型

- 标题 / 活动主题占上方或左上。
- 模特 / 场景占中部或右侧跨格。
- 产品卡占右侧和下排。
- 适合有模特素材、产品图和转化入口的综合型海报。

## 产品摆放规则

- 产品卡必须保持统一尺寸和统一信息层级。
- 下排三格适合三款核心产品并列。
- 右列适合重点产品或补充产品。
- 产品卡内部建议按“货号 / 产品图 / 产品名 / 信息条”排列。
- 如果运营需求没有提供价格、权益、折扣、券包文案，信息条只能作为无文字结构或视觉分隔区。

## 模特素材运用规则

- 模特可以跨格，但脸部、鞋服主体和关键动作不得被格线切断。
- 模特动作要服务品类表达，不得只做无意义摆拍。
- 模特区与产品卡区要分区清楚。
- 不复用参考素材原人物。

## 场景与背景延展规则

- 人物场景可跨格延展，产品卡背景通常保持独立干净。
- 大场景跨格时，关键主体避开九宫格分隔线。
- 背景延展只服务结构，不复用参考图原背景和光影。

## 二维码区域规则

- 二维码必须独占一个完整格。
- 常用位置：中心格、中右格、下中格、中左格。
- 二维码区域要有干净留白和安全边距。
- 正式发布二维码建议后期替换，不建议 AI 直接生成可扫二维码。

## 裁切规则

- 九宫格内部可以有分隔线，但不能切坏人物脸部、产品主体、二维码和主标题。
- 每格需要在手机端可读。
- 跨格素材必须先确认主体位置，再安排文字和产品卡。

## 禁止事项

- 不跨宫格类型套用本规则。
- 不照搬原图品牌。
- 不照搬原图配色。
- 不照搬原图字体。
- 不照搬原图背景。
- 不照搬原图人物。
- 不照搬原图文案。
- 不照搬原图光影。
- 不照搬原图装饰元素。
- 不在最终提示词中写本地路径。
