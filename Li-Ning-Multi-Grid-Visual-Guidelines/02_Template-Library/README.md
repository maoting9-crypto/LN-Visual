# Template Library

## 新定义

`02_Template-Library` 是“参考图信息布局与模块关系模板库”。

它不保存、不复用、不要求照搬参考图的视觉风格。它只从外部参考图库中的图片提取可复用的结构规则。

外部参考图库路径：

```text
G:\Reference-Library\01_Multi-grid-reference
```

## 只沉淀的内容

模板库只沉淀参考图的结构布局，包括：

- 宫格结构。
- 模块位置。
- 信息层级。
- 产品位置。
- 二维码位置。
- 每格功能。
- 主标题位置。
- 权益 / 价格位置。
- 文字区域位置。
- 整体拼接关系。

## 禁止沉淀的内容

模板库不记录、不复用、不要求照搬：

- 原图风格。
- 原图配色。
- 原图背景。
- 原图人物。
- 原图光影。
- 原图品牌气质。
- 原图装饰元素。

## 使用原则

参考图结构是强约束，视觉风格可重塑。

最终海报必须严格保持参考图的信息布局、模块关系、产品位置、二维码位置、文字区域位置和信息层级结构；但最终视觉风格应结合李宁品牌规范、活动主题、品类特点和用户需求重新设计。

## 参考图强锁定模式

如果用户启用强锁定模式，模板必须完全继承参考图的结构层。

固定结构层包括：

- 布局结构。
- 模块关系。
- 各区块位置。
- 产品摆放逻辑。
- 二维码位置。
- 信息层级。
- 拼接关系。

可变视觉层包括：

- 配色。
- 字体设计。
- 图片内元素内容。
- 整体风格调性。
- 人物素材。
- 背景。

强锁定模式下，风格不继承，结构必须继承。模板匹配成功后，输出时必须明确区分“固定结构层”和“可变视觉层”。
