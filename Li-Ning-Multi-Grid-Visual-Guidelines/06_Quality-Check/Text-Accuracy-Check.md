# Text Accuracy Check

## 文案唯一来源检查

最终画面中出现的所有中文文案、数字信息、价格信息、权益信息、折扣信息、券包信息、促销标签、产品名、货号，必须来自运营需求表或用户明确提供的内容。

## 禁止项

不得自行新增、改写、扩写或猜测：

- 主标题
- 副标题
- 权益文案
- 价格文案
- 折扣文案
- 券包文案
- 促销标签
- 产品名
- 货号
- 行动引导

## 参考素材文案检查

参考素材中的文案不能当成当前任务文案使用。

参考素材中的价格区、权益条、促销标签区，只能作为信息模块位置参考，不能复用原文字。

如果运营需求没有提供对应文案，最终提示词只能写：

```text
保留该结构位置，但不生成任何文字内容。
```

## 不合格标准

出现以下情况，检查结果判定为不合格：

- 出现需求中没有的文案。
- 出现需求中没有的价格。
- 出现需求中没有的权益。
- 出现需求中没有的券包、折扣、促销标签。
- 把参考素材里的文字当成当前任务文案。
- 把品牌规范中的通用词当成运营文案。
- 没有标记“需求未提供”。
- 缺失内容影响输出时，没有提醒用户确认。
