# Reference Match Check

## 检查定位

本检查用于确认多宫格任务是否正确学习并保持了参考素材的布局规律和位置关系。

参考素材用于学习布局规则，不是学习风格。

## 布局保持检查

必须检查是否保持：

- 整体宫格结构。
- 每格功能关系。
- 主标题区位置。
- 副标题区位置。
- 产品区位置。
- 产品标签位置。
- 鞋品摆放位置。
- 模特 / 人物素材位置。
- 场景素材使用位置。
- 二维码区域位置。
- 信息层级结构。
- 背景延展关系。
- 裁切规则。

## 不允许自由改变检查

以下任一情况出现，检查结果判定为不合格：

- 自由改变版式骨架。
- 改变宫格结构。
- 改变每格功能。
- 改变产品卡排列方式。
- 移动二维码位置。
- 移动标题区位置。
- 移动产品标签位置。
- 改变模特和产品的相对关系。
- 改变信息层级结构。
- 把六宫格参考图改造成另一种六宫格结构。
- 把局部二维码区域放大成完整独立格，除非参考素材本身就是完整独立格。
- 把产品卡结构重新设计成另一种产品展示方式。

## 允许变化检查

允许根据李宁品牌需求变化：

- 配色。
- 字体设计。
- 产品内容。
- 人物素材。
- 背景素材。
- 节日元素。
- 品牌视觉风格。
- 场景氛围。

## 二维码检查

- 如果参考素材中二维码在第 1 格内部，最终是否仍在第 1 格内部。
- 如果参考素材中二维码只占某一格局部区域，最终是否仍为局部区域。
- 是否错误默认让二维码独占完整格。
- 是否错误移动二维码到其他格子。
- 是否只在对应位置标注“此处二维码区域”。
- 是否没有生成真实二维码。
- 如果需求没有提供二维码下方说明文字，是否没有自动生成。

## 最终提示词检查

最终提示词不得出现：

- 参考 G:
- 按本地路径图片生成
- 自由重构版式
- 随意调整二维码位置
- 重新设计产品卡排列方式

最终提示词应表达为：

```text
从多宫格参考素材中学习并保持布局规律和位置关系，再结合李宁品牌需求重塑视觉风格。
```
