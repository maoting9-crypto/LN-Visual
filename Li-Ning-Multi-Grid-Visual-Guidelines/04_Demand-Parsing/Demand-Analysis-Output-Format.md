```text
- 活动主题：
- 品类：

- 副标题：
- 权益信息：
- 价格信息：
- 产品信息：
- 货号：
- 人物要求：
- 场景要求：
- 风格要求：
- 二维码要求：
- 输出平台：
- 备注：

【需求未提供内容】
- 项目名称：需求未提供
- 使用场景：需求未提供
- 宫格形式：需求未提供
- 活动主题：需求未提供
- 品类：需求未提供

- 副标题：需求未提供

- 风格要求：需求未提供
- 二维码要求：需求未提供
- 输出平台：需求未提供

【需要用户确认内容】
- 会影响最终提示词输出的缺失项：需要用户确认
```

## 禁止输出

不得输出：

- Codex 自动补全
- 自动补全理由
- 建议使用
- 推测为
- 可理解为
- 推荐补充
- 根据李宁规范判断为

## 说明

需求分析只负责读取原文，不负责创作需求。李宁品牌规范只用于后续视觉风格判断，不能用于补写运营需求。
