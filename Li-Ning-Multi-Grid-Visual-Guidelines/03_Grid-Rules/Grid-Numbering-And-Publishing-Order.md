## 二维码区域与编号规则

1. 二维码区域不改变原有格子编号。
2. 如果二维码只是某个格子内的局部模块，该格子仍按原编号发布。
3. 不允许因为二维码区域存在而重新编号或调整发布顺序。
4. 不允许因为需求中需要二维码，就新增一个参考图中不存在的独立二维码格。
5. 二维码具体位置、大小、占比、所在格子或所在模块，必须以用户提供的参考图为准。

## 发布顺序

默认按编号从小到大上传；如平台排序规则不同，发布前必须再次检查预览。
