# Cropping And Export Rules

## 基础切图规则

- 默认 AI 出图阶段输出一张完整多宫格海报，不默认拆成多张单格图片。
- 本文件只用于发布、上传或用户明确要求切图时参考。
- 九宫格整体大图需按 3x3 均分。
- 六宫格需按 2x3 或 3x2 均分。
- 四宫格需按 2x2 均分。
- 重要文字、Logo、二维码区域、产品主体不得被切割线裁断。
- 连续背景型需要检查相邻格衔接。

## 二维码区域裁切与导出规则

1. 如果二维码区域位于格子边缘或跨模块位置，裁切时必须保证“此处二维码区域”文字完整可见。
2. 不允许在切图时把二维码占位区域裁掉。
3. 如果二维码区域只是局部模块，不得因为切图而放大成完整格。
4. 二维码区域位置必须和参考图的原始布局关系一致。
5. 如果需求明确需要二维码，导出图中只在参考图对应位置保留“此处二维码区域”，不生成真实二维码图案。
6. 如果需求明确不需要二维码，则导出图中不保留“此处二维码区域”。
7. 如果需求没有明确是否需要二维码，则标记为“需求未提供”，不要自动添加二维码区域。
