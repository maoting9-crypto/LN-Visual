# Cropping And Export Rules

## 基础切图规则
- 默认 AI 出图阶段输出一张完整多宫格海报，不默认拆成多张单格图片。
- 本文件只用于发布、上传或用户明确要求切图时参考。
- 九宫格整体大图需按 3x3 均分。
- 六宫格需按 2x3 或 3x2 均分。
- 四宫格需按 2x2 均分。
- 重要文字、Logo、二维码、产品主体不得跨切割线。
- 连续背景型需要检查相邻格衔接。
