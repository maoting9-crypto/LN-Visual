## 九宫格共性学习结果

九宫格素材主要沉淀出 5 类布局规则：

1. 标题 + 权益 + 二维码 + 产品矩阵型
2. 人物 / 场景跨格 + 产品卡型
3. 中心二维码 / 中心主题 + 周围产品型
4. 全产品矩阵型
5. 主题区 + 模特区 + 产品区混合型

九宫格常见模块关系：

- 左上、上中或中心格承担主标题 / 主题入口。
- 右上或上方局部承担权益 / 价格 / 活动信息模块。
- 中心、中右、下中常用于二维码独占格。
- 下排常用于 3 个产品卡片并列展示。
- 人物或场景常跨 2-4 个格子形成连续画面。
- 产品卡通常保持统一结构：货号 / 产品图 / 产品名 / 信息条。

九宫格裁切规则：

- 每格必须能独立阅读，但整体仍是一张完整 1:1 大图。
- 跨格人物、跨格背景、跨格场景要避开关键脸部、产品主体、二维码和主标题裁切线。
- 二维码必须独占完整格，不跨格，不压住产品、人物或标题。

## 六宫格共性学习结果

六宫格素材主要沉淀出 3 类布局规则：

1. 左侧主题 + 右侧产品矩阵型
2. 主题 / 二维码 + 多产品卡型
3. 节日主题区 + 产品 / 人物组合型

六宫格常见模块关系：

- 画面多为横向 3x2 或 2x3 结构。
- 一侧用于主题与引导模块，另一侧用于产品矩阵或人物 / 产品组合。
- 二维码适合放在左下、右下、下中或侧边完整格。
- 产品卡应统一尺寸、统一底部信息层级。

六宫格裁切规则：

- 横向六宫格需要保证左右视觉重量平衡。
- 主题模块不可被产品卡挤压。
- 二维码仍需独占完整格。
