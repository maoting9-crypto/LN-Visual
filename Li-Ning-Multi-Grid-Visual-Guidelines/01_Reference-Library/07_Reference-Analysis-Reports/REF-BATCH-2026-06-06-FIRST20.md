## 新增模板

### MG-9G-QR-PROMO-MATRIX-001

新增理由：现有 `MG-9G-PRODUCT-MATRIX-001` 只描述产品矩阵，但本批 1:1 朋友圈图大量出现二维码、价格权益、扫码引导和促销货盘组合，因此需要一个更贴近运营实战的模板。

### MG-4G-VERTICAL-SOCIAL-PROMO-001

新增理由：本批竖版社群图不属于标准 3x3 九宫格，但具备稳定的“主视觉 + 产品货盘 + 二维码”结构，可转译为四宫格系列或单张社群长图，适合沉淀为社群促销模板。
