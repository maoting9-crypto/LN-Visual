## 重要原则

参考素材只用于学习“布局原理和素材运用规则”，不用于照搬画面。

只学习：

- 宫格结构
- 信息模块关系
- 产品摆放规则
- 元素摆放规则
- 模特素材运用规则
- 场景素材运用规则
- 背景延展规则
- 二维码区域规则
- 裁切规则

不学习、不复用：

- 原图品牌
- 原图配色
- 原图字体
- 原图背景
- 原图人物
- 原图文案
- 原图光影
- 原图装饰元素
- 原图具体视觉风格
