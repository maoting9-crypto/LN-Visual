# 李宁朋友圈多宫格视觉规范项目 V1.0

本项目是“李宁朋友圈多宫格项目执行系统 V1.0”，用于把运营需求、参考图、产品图、素材文件和旧项目中的李宁品牌 AI 视觉规范，转化为可直接用于 AI 出图的多宫格提示词。

## 项目目标
- 自动解析运营需求。
- 自动补全缺失信息，并单独标注补全项和理由。
- 匹配多宫格模板。
- 默认输出一张完整的多宫格海报提示词。
- 输出内部格子功能规划，用于说明完整海报里的版式分工。
- 输出简洁检查表。

## 输出原则
- 九宫格、六宫格、四宫格、三宫格和双图组合，默认都按“一张完整多宫格海报”生成提示词。
- 内部格子只做版式规划，不默认拆成每格单独生图。
- 只有用户明确要求切图、单格导出或分别生成时，才额外输出每格独立提示词。
- 如果海报需要二维码，二维码必须单独占用一个完整格子，并预留真实二维码后期替换位置。

## 必须调用的旧项目
```text
E:\LN-Visual\Li-Ning-Brand-AI-Visual-Guidelines
```

本项目负责多宫格执行；旧项目负责李宁品牌调性、字体判断、文案入画、品类规范、活动规范、节日规范和品牌禁区。

## 第一阶段范围
只做规范、Skill、模板规则和提示词输出系统；不写自动切图脚本，不接图片 API，不运行生图。
