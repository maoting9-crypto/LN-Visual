# Output Prompt Rules

## 默认输出

本项目默认输出“整张完整多宫格海报提示词”。

九宫格任务默认最终只生成一张完整 1:1 方形海报大图，内部为 3x3 九宫格结构。

默认输出内容：

- 整张完整多宫格总提示词
- 内部格子功能规划
- 二维码 / Logo / 文字 / 产品位置说明
- 负向限制词
- 简洁检查表

## 不默认输出

默认不输出每格独立生图提示词。

只有用户明确要求“每格单独生成”“导出 9 张”“切图发布”时，才额外输出每格独立提示词。

## 平台适配

用户指定平台时，再输出 ChatGPT Image、即梦、Midjourney、可灵、Stable Diffusion / ComfyUI 适配版。
