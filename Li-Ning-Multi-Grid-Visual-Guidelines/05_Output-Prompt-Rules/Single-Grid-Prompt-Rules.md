# Single Grid Prompt Rules

## 默认不输出单格提示词

本项目默认输出一张完整多宫格海报提示词。

只有用户明确要求以下内容时，才输出单格提示词：

- 切图
- 每格单独生成
- 导出 9 张
- 导出 6 张
- 每格分别给提示词

## 单格规划用途

内部格子功能规划只用于说明完整海报内部的版式分区，不代表要拆成多张图片生成。

## 单格规则来源

单格规则来自多宫格参考素材中学习出的布局关系，不来自原图风格。

单格提示词不得包含：

- 参考 G:
- 按本地路径图片生成
- 照搬原图风格
- 完全复刻参考图视觉风格

## 文案规则

单格内出现的文字必须来自运营需求表或用户明确提供内容。

未提供文字时，只能写“预留文字区”，不得擅自生成文案。
