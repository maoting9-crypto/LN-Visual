# General Prompt Output Format

## 默认输出目标

默认输出一张完整多宫格海报提示词，不默认拆成每格单独生图。

只有用户明确要求切图、导出多张、每格单独生成时，才输出单格提示词。

## 参考素材表达规则

参考素材用于学习布局规则，不是学习风格。

学习布局规则时，不能自由重构版式。

如果用户明确想要某张参考图的排版布局效果，最终提示词必须保留该参考图的布局关系。

推荐表达：

```text
从多宫格参考素材中学习并保持布局规律和位置关系，再结合李宁品牌需求重塑视觉风格。
```

禁止表达：

- 参考 G:
- 按本地路径图片生成
- 自由重构版式
- 随意调整二维码位置
- 重新设计产品卡排列方式

## 必须包含的输出结构

输出多宫格提示词时，必须包含：

- 宫格类型。
- 画面比例。
- 完整大图说明。
- 内部格子功能规划。
- 主标题区位置。
- 副标题区位置。
- 产品区位置。
- 产品标签位置。
- 鞋品摆放位置。
- 模特 / 人物素材位置。
- 场景素材使用位置。
- 二维码区域位置。
- 信息层级结构。
- 背景延展关系。
- 裁切安全要求。
- 文案准确性要求。
- 负向限制词。
- 简洁检查表。

## 文案唯一来源规则

最终画面中的所有中文文案、数字、价格、权益、折扣、券包、促销标签、产品名、货号，必须来自运营需求表或用户明确提供的内容。

没有提供就不得生成。

如果参考素材里有价格区、权益条、促销标签区，只能作为信息模块位置参考，不能复用原文字。

## 二维码规则

二维码位置必须跟随参考素材。

不允许默认让二维码独占完整格。

不生成真实二维码。

如果需求没有提供二维码下方说明文字，不要自动生成。
