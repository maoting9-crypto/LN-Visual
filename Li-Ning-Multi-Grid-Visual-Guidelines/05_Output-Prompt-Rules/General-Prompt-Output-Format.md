# General Prompt Output Format

## 默认输出格式

最终输出必须分为两层：内部分析层和外部生图提示词层。

```text
【需求解析】
【模板匹配】
【内部分析层：参考图结构分析】
【外部生图提示词层：完整多宫格海报总提示词】
【内部格子功能规划】
【二维码 / Logo / 文字 / 产品位置说明】
【负向限制词】
【简洁检查表】
```

## 内部分析层

内部分析层可以记录：

- 本地参考图路径。
- 匹配到的模板编号。
- 参考图文件名。
- 参考图结构分析。
- 候选参考图列表和判断理由。

内部分析层只保存在项目文件、分析报告或工作说明中，不直接复制到外部 AI 生图平台。

## 外部生图提示词层

外部生图提示词层必须满足：

- 不允许出现本地路径。
- 不允许写“参考某文件”“参考本地路径”“参考上图”“按照本地图片”。
- 必须把参考图内容转写成完整的文字描述。
- 必须可直接复制到 ChatGPT Image、即梦、Midjourney、可灵、Stable Diffusion / ComfyUI 等外部 AI 平台使用。
- 必须表达为：保持参考图的信息布局和模块关系，但重新设计为李宁品牌视觉风格。

## 需求原文使用规则

- 最终提示词只能使用需求表或用户明确提供的文案。
- 不允许添加需求中没有的中文文案。
- 不允许添加需求中没有的价格、权益、券包、折扣、促销标签。
- 不允许添加需求中没有的产品名、货号。
- 不允许添加需求中没有的人物设定。
- 不允许添加需求中没有的场景设定。
- 如果需求没有提供某项内容，最终提示词中不能擅自补写。
- 如果必须保留参考图中的某个结构区域，但需求没有提供文字内容，只能写：“保留该结构位置，但不生成任何文字内容。”
- 不允许写“可添加促销文字”“可补充价格信息”“可加入优惠标签”“自动补充产品卖点”“自动生成活动口号”。

## 强锁定模式最终提示词表达规则

当参考图处于强锁定模式时，最终提示词必须明确说明：

- 这是基于参考图原有版式结构进行的视觉重设计。
- 严格保持参考图的整体构图、模块位置、产品位置、二维码位置、文字区域位置和信息层级结构。
- 只允许修改配色、字体设计、图片内元素、风格调性、人物素材和背景。
- 不允许重新设计版式。
- 不允许改变各模块之间的相对位置关系。
- 不允许调整产品卡的排列逻辑。
- 不允许改变二维码位置。
- 不允许改变主标题区和其他文字区的位置。
- 如果需求中没有提供价格 / 权益 / 券包文案，则只保留其结构位置，不生成任何文字。

最终生图提示词必须包含：

1. 画面比例与宫格形式。
2. 参考图结构转写后的版式描述。
3. 每格功能分布。
4. 主标题区位置。
5. 产品区位置。
6. 二维码区位置。
7. 文字层级。
8. 李宁品牌视觉风格。
9. 需求文案。
10. 产品要求。
11. 负向限制。

如果参考图中有价格条、黄色条、权益条或底部信息条，最终生图提示词必须说明这些区域是结构位置参考，不是文案内容参考。

如果运营需求没有提供价格 / 权益 / 折扣 / 券包文案，最终生图提示词必须写明：

```text
保留产品卡底部的信息承载条形区域，但不生成任何价格、权益、折扣或促销文字，除非运营需求明确提供。
```

如果运营需求提供了价格或权益，最终生图提示词必须写明：

```text
产品卡底部信息条中仅显示需求表提供的原文：xxx。
```

最终生图提示词必须明确：

- 严格保持布局结构。
- 严格保持模块位置。
- 严格保持产品区域。
- 严格保持二维码区域。
- 严格保持文字层级。
- 不照搬原图配色。
- 不照搬原图背景。
- 不照搬原图人物。
- 不照搬原图风格。
- 结合李宁品牌调性重新设计。
- 强锁定模式下，只重塑视觉风格，不重构版式结构。

禁止出现：

- 本地路径。
- “参考此路径”。
- “参考 G:”。
- “参考上图”。
- “参考文件”。
- “按照本地图片”。
- “照搬原图风格”。
- “完全复刻参考图视觉风格”。
- “重新设计版式”。
- “调整模块位置”。
- “重新排列产品卡”。
- 任何外部 AI 平台无法读取的信息。
- 需求表没有提供的“预估到手价”“到手价”“券后价”“限时优惠”“立减”“满减”“折扣”“券包”“爆款”“好价”“福利”“促销标签”“价格数字”“货号”“产品名称”。
- “可添加促销文字”“添加优惠信息”“价格区写活动价”“自然补充促销标签”等模糊表达。

## 九宫格输出要求

九宫格默认必须写明：

```text
生成一张完整的 1:1 方形九宫格海报大图，画面内部为 3x3 网格布局，最终只输出单张完整图片，不要拆成 9 张单独图片。
```

## 单格提示词规则

默认不输出单格独立生图提示词。

只有用户明确要求单格导出或切图时，才追加：

```text
【每格独立提示词】
```
