# Platform Adaptation Rules

## 通用原则

默认先输出通用中文提示词。

只有用户指定平台时，再输出对应平台适配版本。

## 本地路径规则

所有平台版本都不得包含本地路径。

禁止写：

- 参考 G:
- 按本地路径图片生成
- 参考本地图片

应写成：

```text
从多宫格参考素材中学习布局原理和素材运用规则，并转写为外部 AI 平台可理解的完整描述。
```

## 平台建议

- ChatGPT Image / 即梦 / 可灵：可以尝试生成完整多宫格海报，但中文、价格、货号和二维码建议后期复核。
- Midjourney：不建议强制生成复杂中文，建议预留文字区域，后期排版。
- Stable Diffusion / ComfyUI：建议画面与文字分离，文字后期排版。
- 视频平台：多宫格规则可作为首帧、封面或分镜结构参考。

## 二维码

不建议任何平台直接生成真实可扫二维码。

应预留二维码完整格，正式发布时后期替换。
