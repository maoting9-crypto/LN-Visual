# Platform Adaptation Rules

## 默认
默认只输出通用中文提示词。

## 外部平台通用要求

所有平台适配版提示词都必须是自包含提示词：

- 不允许出现本地路径。
- 不允许写“参考本地文件”“参考上图”“参考某路径”。
- 必须把参考图结构转写成画面布局文字。
- 复制到 ChatGPT Image、即梦、Midjourney、可灵、Stable Diffusion / ComfyUI 后，平台不需要读取本地文件也能理解画面。
- 必须表达为：保持参考图的信息布局和模块关系，但重新设计为李宁品牌视觉风格。
- 必须明确严格保持布局结构、模块位置、产品区域、二维码区域和文字层级。
- 必须明确不照搬原图配色、原图背景、原图人物、原图风格、原图光影和原图装饰元素。
- 必须结合李宁品牌调性、活动主题、品类特点和用户需求重新设计视觉风格。

## 强锁定模式平台适配要求

当参考图进入强锁定模式时，所有平台版本都必须表达：

- 基于参考图原有版式结构进行视觉重设计。
- 严格保持整体构图、宫格结构、模块位置、产品位置、二维码位置、文字区域位置和信息层级结构。
- 只允许修改配色、字体设计、图片内元素内容、整体风格调性、人物素材和背景。
- 不重新设计版式。
- 不改变模块相对位置。
- 不调整产品卡排列逻辑。
- 不改变二维码位置。
- 不改变主标题区和其他文字区位置。
- 没有价格 / 权益 / 券包文案时，只保留结构位置，不生成文字。

## 平台适配禁止语

所有平台适配版最终提示词禁止出现：

- 本地路径。
- `参考 G:`。
- `参考某文件`。
- `参考本地图片`。
- `照搬原图风格`。
- `完全复刻参考图视觉风格`。
- `重新设计版式`。
- `调整模块位置`。
- `重新排列产品卡`。
- 任何需要外部 AI 平台读取本地文件才能理解的信息。

## 平台策略
- ChatGPT Image / 即梦 / 可灵：可尝试主标题入画，但需提醒中文风险。
- Midjourney：不建议复杂中文，建议预留文字区。
- Stable Diffusion / ComfyUI：建议画面与文字分离。
- 视频平台：文字可作为首帧或字幕系统。
