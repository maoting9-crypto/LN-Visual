# Full Grid Prompt Rules

## 默认规则

多宫格任务默认生成“一张完整多宫格海报”的最终提示词。

不要默认拆成每格单独生图。

## 多宫格参考素材学习规则｜修正版

参考素材用于学习布局规则，不是学习风格。

学习布局规则时，不能自由重构版式。如果用户明确想要某张参考图的排版布局效果，Codex 必须保留该参考图的布局关系。

参考素材必须作为完整海报的版式骨架处理。最终提示词不能让外部 AI 重新设计布局，只能让外部 AI 在既有结构上替换视觉层。

最终完整海报提示词必须先写：

```text
以多宫格参考素材为版式骨架，严格保持宫格结构、模块位置、产品卡排列、二维码位置、信息层级和裁切关系，只替换配色、字体、产品内容、人物素材、背景素材、节日元素、品牌视觉风格和场景氛围。
```

最终完整海报提示词必须准确描述并保持：

- 整体宫格结构。
- 每格功能关系。
- 主标题区位置。
- 副标题区位置。
- 产品区位置。
- 产品标签位置。
- 鞋品摆放位置。
- 模特 / 人物素材位置。
- 场景素材使用位置。
- 二维码区域位置。
- 信息层级结构。
- 背景延展关系。
- 裁切规则。
- 跨格连续人物 / 场景 / 产品大画面的拼接关系。

## 跨格连续场景硬规则

如果参考素材中多个格子共同组成一张连续人物场景、连续背景场景或连续产品穿着场景，最终提示词必须明确写成：

```text
第 X / X / X / X 格共同组成一张连续完整场景，这些格子不是独立小图，而是同一张人物 / 背景 / 产品穿着大画面被九宫格分隔线切开。
```

必须保持：

- 人物身体、姿态、视线或局部肢体在相邻格之间的连续关系。
- 背景材质、透视、光影和空间方向在相邻格之间的连续关系。
- 鞋品、服装或产品穿着细节在相邻格之间的自然延展。
- 白色宫格分隔线只切分画面，不改变场景连续性。

不允许：

- 把连续场景描述成每格独立场景。
- 逐格重新生成不同人物、不同背景、不同光影。
- 把跨格人物或鞋品局部拆散成无关系的画面。
- 只写“人物氛围区”“鞋品细节区”，却不说明它们属于同一张连续拼接场景。

## 允许调整

只允许根据李宁品牌需求重新调整：

- 配色。
- 字体设计。
- 产品内容。
- 人物素材。
- 背景素材。
- 节日元素。
- 品牌视觉风格。
- 场景氛围。

## 禁止调整

不允许自由改变：

- 版式骨架。
- 宫格结构。
- 每格功能。
- 产品卡排列方式。
- 二维码位置。
- 标题区位置。
- 产品标签位置。
- 模特和产品的相对关系。
- 信息层级结构。
- 背景跨格延展关系。
- 裁切关系。

不允许在最终提示词中使用会让 AI 重新排版的表达，例如：

- 重新设计版式。
- 优化布局。
- 自由排版。
- 创意重组。
- 参考风格重新生成。
- 根据画面需要调整二维码位置。

## 二维码区域规则

- 如果参考素材中二维码在第 1 格内部，最终也必须在第 1 格内部。
- 如果参考素材中二维码只占某一格的局部区域，最终也只能保留为局部区域。
- 不允许默认让二维码独占一个完整格子。
- 不允许把二维码移动到其他格子。
- 需要二维码时，只在对应位置标注“此处二维码区域”。
- 不生成真实二维码。
- 如果需求没有提供二维码下方说明文字，不要自动生成。

## 本地路径脱敏

最终提示词不得包含：

- `G:\Reference-Library`
- `参考 G:`
- `按本地路径图片生成`

应改写为：

```text
以多宫格参考素材为版式骨架，严格保持宫格结构、模块位置、产品卡排列、二维码位置、信息层级和裁切关系，只根据李宁品牌需求替换配色、字体、产品内容、人物素材、背景素材、节日元素、品牌视觉风格和场景氛围。
```

## 价格 / 权益区

如果运营需求没有提供价格、权益、折扣、券包文案：

- 不生成任何价格类文字。
- 不生成任何优惠类文字。
- 不写“预估到手价”“券后价”“限时优惠”等占位文字。
- 如果结构需要保留该区域，只能描述为无文字信息承载区、空白信息条或视觉分隔色块。
