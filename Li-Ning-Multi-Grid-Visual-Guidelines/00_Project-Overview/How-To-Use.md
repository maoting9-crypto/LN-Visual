# How To Use

## 使用步骤
1. 把运营需求发给 Codex，或放入 `01_Reference-Library/06_Operation-Briefs/`。
2. 把参考图放入 `01_Reference-Library/01_Raw-References/`。
3. 把产品图放入 `01_Reference-Library/03_Product-Images/`。
4. 调用 `li-ning-multi-grid-director` Skill。
5. 先输出需求解析，再输出模板匹配、总提示词、单格提示词和检查表。

## 调用示例
```text
请调用 li-ning-multi-grid-director Skill，解析这个李宁朋友圈多宫格需求，并输出整组总提示词、每格提示词和简洁检查表。
```
