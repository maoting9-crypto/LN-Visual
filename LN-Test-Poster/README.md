本项目不作为正式规范库使用。

## 使用原则

- 如果测试李宁品牌海报，需要调用：

```text
E:\LN-Visual\Li-Ning-Brand-AI-Visual-Guidelines
```

- 如果测试朋友圈多宫格海报，需要调用：

```text
E:\LN-Visual\Li-Ning-Multi-Grid-Visual-Guidelines
```

- 本项目可以保持轻量，不需要独立 Skill。
- 输出内容仅用于测试和验证，不作为最终规范沉淀。
